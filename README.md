# ReplaceG2O

This is an EXPERIMENTAL Chrome extension to replace embedded Google map to OpenStreetMap.

No maintainance, maybe.

[![Example](http://img.youtube.com/vi/N_0H6LlOFLw/0.jpg)](http://www.youtube.com/watch?v=N_0H6LlOFLw)
